# Лабораторная работа №7–1 "Алгоритмы сортировки и поиск ближайшей пары точек"
---
### 💻 Как собрать и запустить проект
```bash
rm -rf build && cmake -B build && cmake --build build && ./build/app
```
---
### 🎯 Цель работы
1. Реализовать и сравнить по времени работы три алгоритма сортировки:
   - Сортировка **выбором** (SelectionSort);
   - Сортировка **пузырьком** (BubbleSort);
   - Сортировка **слиянием** (MergeSort).
2. Реализовать **последовательный поиск** с использованием ключа-«сторожа».
3. Реализовать задачу поиска **пары ближайших точек**:
   - методом **грубой силы** (Brute Force);
   - методом **декомпозиции (разделяй и властвуй)**.
4. Сравнить точность и время выполнения различных алгоритмов.
